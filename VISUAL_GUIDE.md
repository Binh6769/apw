## рҹҺҜ User Flows

### Flow 1: Auto-Recording Image Views

```
User browses feed
    вҶ“
Clicks image 1
    вҶ“
PinDetail page loads
    вҶ“
[Image auto-recorded to database]
    вҶ“
User views details, reads comments
    вҶ“
Goes back to feed
    вҶ“
Clicks image 2
    вҶ“
[Image auto-recorded]
    вҶ“
Later: User goes to History
    вҶ“
Sees both images in grid вң“
```

### Flow 2: Create Album & Add Images

```
User clicks menu вҶ’ "Photo Albums"
    вҶ“
Clicks "Create Album"
    вҶ“
Enters name: "Favorites"
    вҶ“
Album created
    вҶ“
Goes to History
    вҶ“
Clicks "..." on image
    вҶ“
Selects "Add to Album"
    вҶ“
Chooses "Favorites"
    вҶ“
[Image added to album]
    вҶ“
Goes back to Albums
    вҶ“
Sees image in "Favorites" album вң“
```

### Flow 3: Time-Based Deletion

```
User goes to History
    вҶ“
Clicks "By Day" filter
    вҶ“
Date picker appears
    вҶ“
Selects: Yesterday
    вҶ“
Grid shows only yesterday's images
    вҶ“
Clicks "Delete Period"
    вҶ“
Confirmation dialog: "Delete all images from this day?"
    вҶ“
User confirms
    вҶ“
All images from yesterday deleted
    вҶ“
Grid now shows images from other days вң“
```

### Flow 4: Search & Save

```
User goes to History
    вҶ“
Searches: "sunset"
    вҶ“
Results filtered in real-time
    вҶ“
Sees sunset image
    вҶ“
Clicks "Save" button
    вҶ“
[Button changes from Red to Black]
    вҶ“
Toast: "Saved to Profile"
    вҶ“
Image added to saved pins
    вҶ“
User's profile now shows image вң“
```

## рҹ“Ҡ Database Schema

### image_history Table
```
id (UUID, PK)
в”ңв”Җ user_id (UUID, FK) в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җв”ҖвҶ’ auth.users
в”ңв”Җ image_id (TEXT)
в”ңв”Җ image_url (TEXT)
в”ңв”Җ image_title (TEXT)
в”ңв”Җ image_width (INT)
в”ңв”Җ image_height (INT)
в”ңв”Җ image_color (TEXT)
в”ңв”Җ image_description (TEXT)
в”ңв”Җ source (TEXT) в”Җв”Җв”Җв”Җв”Җв”Җв”Җв”Җ ['unsplash', 'pin', 'external']
в”ңв”Җ viewed_at (TIMESTAMP)
в””в”Җ created_at (TIMESTAMP)

Indexes:
вҖў (user_id, viewed_at DESC)
вҖў (user_id)

RLS Policies:
вҖў SELECT: WHERE user_id = auth.uid()
вҖў INSERT: WITH CHECK user_id = auth.uid()
вҖў DELETE: WHERE user_id = auth.uid()
вҖў UPDATE: WHERE user_id = auth.uid()
```

```
id (UUID, PK)

в”ңв”Җ is_public (BOOLEAN)

в””в”Җ updated_at (TIMESTAMP)

вҖў (user_id)
вҖў (user_id, created_at DESC)

RLS Policies:

вҖў INSERT: WITH CHECK user_id = auth.uid()
вҖў UPDATE: WHERE user_id = auth.uid()
вҖў DELETE: WHERE user_id = auth.uid()
```
